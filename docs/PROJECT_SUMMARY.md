## 📊 Текущая архитектура

### Двухконтурная система:

```
┌─────────────────────────────────────────┐
│  Бот (ВАШ_ХОСТИНГ.ru)                │
│  • Polling режим                        │
│  • AI-консультанты                      │
│  • SQLite (заказы, корзина)            │
│  • MySQL Client (читает уведомления)   │
└───────────┬─────────────────────────────┘
            │ MySQL Relay (polling 20s)
            ▼
┌─────────────────────────────────────────┐
│  MySQL Database                         │
│  yookassa_notifications table          │
└───────────┬─────────────────────────────┘
            │ HTTPS POST
            ▲
┌───────────┴─────────────────────────────┐
│  Webhook (solidsimple.ru)              │
│  • SSL сертификат                       │
│  • webhook.php                          │
│  • HMAC проверка                        │
│  • Запись в MySQL                       │
└───────────┬─────────────────────────────┘
            │ YooKassa API
            ▼
┌───────────┴─────────────────────────────┐
│  YooKassa Payment System               │
└─────────────────────────────────────────┘
```

### Почему так?
- **YooKassa требует HTTPS** → webhook на solidsimple.ru с SSL
- **Бесплатный хостинг не даёт SSL** → бот отдельно на xsph.ru
- **Нужна связь** → общая MySQL база
